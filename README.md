<h2><b>Requisitos</b></h2>
    
    JAVA 8
    Node 12
    Intellij ou Eclipse

<h2><b>Passos para rodar o projeto</b></h2>

1- Executar <b>git clone https://github.com/MatheusSouza072/tabela-fipe-backend.git</b><br>
2- Abrir projeto em uma IDE, Intellij ou Eclipse <br>
3- Esperar baixar todas as dependências <br>
4- Instalar pluguin do Lombok <br>
5- Executar a classe TabelafipeApplication.java <br>

Observação: O projeto estará rodando na porta 8091
